# Expense Analyzer & Budget Advisor

A Python web app that analyzes your personal expenses and gives budget advice.

## Features
- Upload any CSV or Excel expense file
- Visual breakdown of spending by category and month
- Set custom budgets and instantly see where you're over or under
- Interactive charts built with Plotly

## Tech stack
- Python, pandas, Streamlit, Plotly

## How to run locally
pip install -r requirements.txt
streamlit run app.py
